### STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-03-01 | Tampering | YAML files with lint violations introduced | mitigate | D-04 requires ansible-lint + yamllint pass; pre-commit hooks enforce yamllint --strict on every commit; CI enforces ansible-lint on every PR |
| T-01-03-02 | Denial of Service | Integration tests triggering unintended k8s resource creation | accept | Kind cluster is isolated; test fixtures use test namespaces; no production cluster is used for D-03 regression gate |
| T-01-03-03 | Information Disclosure | kubeconfig or secrets in test output logs | accept | Integration tests use kind cluster with no real credentials; test kubeconfig is generated for the kind cluster only |
| T-01-03-04 | Elevation of Privilege | Orphaned task removal allowing unintended resource persistence | accept | Orphaned task never created the resource it attempted to delete; removal reduces unnecessary k8s API calls, not access scope; soft-fail already prevented any privilege effect |
