       ```
       pick stu5678 docs(ocp_virt_vm): update README for Windows
       squash vwx9012 docs: add argument_specs
       ...
       pick yza3456 fix(ocp_virt_vm): PR #294 review feedback
       squash bcd7890 fix: Secret migration
       squash efg1234 fix: test validation
       ```
